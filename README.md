# Simple Sum Program

## Описание проекта

Это простое приложение для суммирования чисел для первой лабораторной работы по разработке ПО.

## Основные функции

- Суммирование двух чисел через `my_sum(a, b)`
- Умножение двух чисел через `my_mult(a, b)`
- Обработка аргументов командной строки
- Интеграция с PostgreSQL

## Сборка и запуск

```bash
# Сборка
cmake -B build .
cmake --build build

# Запуск
./build/sum 1 2 3 4

# Тестирование
cd build && ctest
```
